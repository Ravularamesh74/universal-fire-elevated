# 🔥 Universal Fire Safety & Service

🌐 **Live Demo:** https://universalfire-safety.netlify.app

---

## 📌 Overview

**Universal Fire Safety & Service** is a modern, responsive web application built to provide comprehensive fire safety solutions. The platform focuses on delivering fire protection services, safety equipment information, and regulatory compliance support for industries, commercial buildings, and residential complexes.

This project demonstrates a real-world business use case with a clean UI, scalable structure, and deployment-ready architecture.

---

## 🚀 Features

### 🧯 Fire Safety Equipment

* Fire Extinguishers (ABC, CO₂, Foam, Water-based)
* Fire Alarm Systems
* Smoke Detectors
* Sprinkler Systems
* Hydrant Systems

### 🛠️ Fire Safety Services

* Installation of fire safety systems
* Maintenance and servicing
* Fire safety audits and inspections
* Emergency support services

### 📄 NOC & Compliance

* Provisional Fire NOC guidance
* Final Fire NOC approval support
* Documentation handling
* Inspection readiness

### 📱 Modern UI/UX

* Fully responsive design
* Smooth navigation
* Professional layout
* Optimized for performance
